// Das Problem

Dein Makler weiß nicht, was ein "Staff Engineer" ist. Dein BU-Vertrag wahrscheinlich auch nicht.

Falsche Berufsgruppe eingetragen

Viele ITler werden als "kaufmännisch" eingestuft. Das bedeutet höhere Prämien oder schlechtere Leistungen im Ernstfall.

Freelancer? Welche Klauseln?

Abstrakte Verweisung, Einkommensnachweis, Gehaltsvergleich: Für Freelancer gelten komplett andere Spielregeln.

Burnout wird systematisch unterschätzt

Psychische Erkrankungen sind die häufigste BU-Ursache in der IT. Einige Gesellschaften schließen sie versteckt aus.
